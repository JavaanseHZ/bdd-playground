# BDD Playground

Playground for Behaviour Driven Development

## Presentation

https://javaansehz.github.io/bdd-playground/

## Backend (Springboot)

Start with: ./gradlew bootRun

## Frontend (Angular)

Start with: ./ng serve

## Run tests
### Backend

./gradlew bddUnitTest

./gradlew bddIntegrationTest

### Frontend

./ng e2e

### Reporting

./gradlew allureReport

./gradlew/cluecumberReports
